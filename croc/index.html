Если Android Deep Link не сработал, настройте открытие ссылок вручную:

  • На странице ➤ Дать / Send нажмите кнопку 🗐.
  • Из Секрета и параметров Посредника сформируется Deep Link и QR-код.
  • В диалоге Deep Link нажмите ⚙️ Deep Link.

Android 12+ (API 31+):

  • Включите 🟢 Открывать поддерживаемые ссылки / Open supported links.
  • В меню ❯ Добавить ссылку отметьте ✅ abakum.github.io и нажмите Добавить.
APP_OPEN_BY_DEFAULT_SETTINGS

Android 11- (API 30-):

  • В меню ⌵ Дополнительно / Advanced -> Открывать по умолчанию / Open by default.
  • В меню Открывать поддерживаемые ссылки / Open supported links выберите Открывать в этом приложении / Open in this app.
APPLICATION_DETAILS_SETTINGS

Открыть ⚙️ Deep Link если установлен Croc, а если не установлен то скачать APK для вашей архитектуры:
